# Collie

AI-powered GitHub repository triage for solo maintainers.

## Install

```bash
pip install collie
```

## Commands

| Command | Description |
|---------|-------------|
| `collie sit <repo>` | Analyze repository issues and PRs |
| `collie bark <repo>` | Post triage comments on open issues and PRs |
| `collie approve <repo> [numbers...]` | Approve issues or PRs |
| `collie reject <repo> <number>` | Reject an issue or PR |
| `collie shake-hands <repo>` | Onboard a new contributor |
| `collie unleash <repo>` | Enable automated triage |
| `collie leash <repo>` | Disable automated triage |
| `collie status <repo>` | Show triage status |
